Welcome to ZekOS

ZekOS is a minimal NASM-based operating system project inspired by MikeOS. It’s built to help you learn how real mode assembly and bootloaders work on x16, x32, and x64 systems.

Dependencies

1: DD For Windows

2: ImDisk Disk manager

3: VM, or QEMU

Why ZekOS?

  • Simple, readable code
  • Works on x16/x32/x64 systems
  • Great for learning NASM and bootloaders
