About ZekOS

ZekOS is a minimal assembly language operating system designed for low-level system software work. It is a fork from MikeOS but built with simplicity and clarity in mind. I am making this for fun but I hope this will because famous. ZekOS is in development and we do not know when we will release it.

Goals of the Project

  • Be a large OS.
  • Run on x16, x32, and x64 systems
  • Be fully open and modifiable

Tech Stack

  • Assembler: NASM
  • Bootloader: Real mode
  • Platform: x16 / x32 / x64
  • Language: Assembly

Made by Zeke

This project is developed by Zeke as a personal learning project and a tool to help others discover how simple OS development can be. If you're interested in contributing or learning, feel free to fork and build your own version!
